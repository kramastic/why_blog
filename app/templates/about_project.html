Проект WHYBLOG

Веб-сайт c функционалом персонального блога

Технологический стек:
  • Python 3.10, FastAPI, PostgreSQL, Celery, Redis, asyncio, Pydantic, CSS, Jinja2
  • Взаимодействие с базой данных - SQLAlchemy, alembic
  • Админка - SQLAlchemy Admin

Функционал:

  • 1. Регистрирация нового пользователя и прохождение аутентификации, выход из своего профиля (исп. JWT)
  • 2. Размещение на персональной странице постов (статей), прикрепление фото к посту
  • 3. Редактирование/удаление поста после публикации
  • 4. Просмотр страниц других пользователей
  • 5. После регистрации на ваш e-mail будет отправлено письмо с подтверждением регистарции (исп. Celery)
  • 6. Полноценный API для взаимодействия с сайтом не только через GUI
  • 7. Админка для удобной работы с базой данных
